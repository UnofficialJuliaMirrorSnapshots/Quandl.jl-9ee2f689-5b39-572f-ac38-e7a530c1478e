Quandl Overview
===============

`Quandl.com <http://www.quandl.com>`__ is a lightweight interface to
over nine million open-source datasets. This package creates an easy
interface to obtain and manipulate these datasets, using TimeArrays or
DataFrames, as well as letting the user search for data.
The Quandl package is an API to the Quandl.com resource for open-source data sets.
